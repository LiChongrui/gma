    Open:RasterOpen

    洛 2022/3/9

     　打开的 栅格数据 具有如下所列的 方法/属性：

    # Bands

    功能：【波段数】。类属性，获取栅格波段数。

    返回：str。

    # Columns

    功能：【列数】。类属性，获取栅格列数（X）。

    返回：int。

    # DataType

    功能：【数据类型】。类属性，获取栅格数据的数据类型。如果为多波段数据，则取所有波段数据类型的众数。

    返回：int。

    # Driver

    功能：【栅格驱动】。获取栅格数据驱动。

    返回：str。

    # GeoTransform

    功能：【仿射变换】。类属性，获取栅格仿射变换。

    返回：tuple。

    # GetBand(Band)

    功能：【获取 Band】。读取某一波段为 gdal.Band。

    参数：

     Band：int。栅格文件的波段数。从 1 开始。

    返回：gdal.Band。

    # GetBandMetadata(Band)

    功能：【波段元数据】。获取某一波段的元数据。

    参数：

     Band：int。栅格文件的波段数。从 1 开始。

    返回：dict。

    # GetBandToArray(Band)

    功能：【转换波段为数组】。将某一波段读取为 Numpy 数组。

    参数：

     Band：int。栅格文件的波段数。从 1 开始。

    返回：array。

    # GetGDALDataset()

    功能：【获取 GDALDataset】。读取栅格数据为 gdal.GDALDataset。

    返回：gdal.GDALDataset。

    # Info

    功能：【栅格信息】。类属性，获取栅格信息。

    返回：dict。

    # Metadata

    功能：【元数据】。类属性，获取栅格元数据。

    返回：dict。

    # NoData

    功能：【无效值】。类属性，获取栅格无效值。

    返回：float||None。

    # Projection

    功能：【坐标系】。类属性，获取栅格坐标系。

    返回：str。

    # Rows

    功能：【行数】。类属性，获取栅格行数（Y）。

    返回：int。

    # ToArray(LeftRow = 0, LeftColumn = 0, RowSize = None, ColumnSize = None, BandList = None)

    功能：【转为数组】。将栅格数据读取为 Numpy 数组。

    可选参数：

     LeftRow = int。需要读取数据的起始行。默认（0）为第一行。

     LeftColumn = int。需要读取数据的起始列。默认（0）为第一列。

     RowSize = int。从起始行开始读取的行数（数据窗口的行宽度）。默认（None）从起始行开始的全部行。

     ColumnSize = int。从起始列开始读取的列数（数据窗口的列宽度）。默认（None）从起始列开始的全部列。

     BandList = int||list。读取数据的波段编号或波段编号列表。默认（None）全部波段（编号从 1 开始）。

    返回：array。
