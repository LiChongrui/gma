## 什么是 GmaGIS?

&emsp;&emsp; GmaGIS 是以 Python 语言为基础，以 PyQT6 为 GUI 底层，以PyQGIS 为地图渲染组件，依托 gma 库构建的桌面端 GIS 软件。

## 基础界面设计

&emsp; &emsp; GmaGIS 基础界面设计参考了 ArcGIS Pro，具有现代化的界面风格。
