## gma.map.plot.**GetWorldDataSource**()<Badge text="1.1.2 +"/> 

---

**功能：** 【获取世界数据资源】。打开内置的低精度世界国家和政区矢量数据资源。

**返回：** `gma.algorithm.core.dataio.DataSource`。

---

**示例：**
```python
from gma.map import plot
WorldDS = plot.GetWorldDataSource()
```
