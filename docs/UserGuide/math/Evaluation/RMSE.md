## gma.math.Evaluation().**RMSE**()

---

**功能：**【RMSE】。均方根误差。

**返回：**`float||array`。

---

**示例：**

```python
import gma
```

*序列（1 维）*

```python
MEA = [15.1, 33, 88, 158.4]
SIM = [0.8, 1.7, 7.8, 7.4]
EVA = gma.math.Evaluation(MEA, SIM)
EVA.RMSE()
```
> \>>> 87.20266624364189
