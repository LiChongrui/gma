## gma.math.Evaluation().**MaxAE**()

---

**功能：**【MaxAE】。最大绝对误差。

**返回：**`float||array`。

---

**示例：**

```python
import gma
```

*序列（1 维）*

```python
MEA = [15.1, 33, 88, 158.4]
SIM = [0.8, 1.7, 7.8, 7.4]
EVA = gma.math.Evaluation(MEA, SIM)
EVA.MaxAE()
```
> \>>> 136.7

*更多维度*
