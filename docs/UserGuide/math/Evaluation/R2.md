## gma.index.Evaluation().**R2**()

---

**功能：**【R2】。决定系数。

**返回：**`float||array`。

---

**示例：**

```python
import gma
```

*序列（1 维）*

```python
MEA = [15.1, 33, 88, 158.4]
SIM = [0.8, 1.7, 7.8, 7.4]
EVA = gma.math.Evaluation(MEA, SIM)
EVA.R2()
```
> \>>> 0.7642034563762724

*更多维度*
