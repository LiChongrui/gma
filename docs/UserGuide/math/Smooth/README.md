## gma.index.**Smooth**(*Data, WindowSize, Times = 1*)

---

**功能：** 【类】。数据平滑（滤波）。

**初始化：**

&emsp;Data：1D `array`。需要平滑的 1 维数据。

&emsp;WindowSize：`int`。平滑窗口大小。必须为正奇数。

&emsp;Times = `int`。平滑次数。默认平滑 1 次。

::: warning 注意

类内 属性/方法 引用前请先初始化 `Smooth` 类！

:::
