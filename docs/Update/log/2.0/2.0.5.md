**&emsp;2、部分函数名调整**：对gma部分内容进行了修改，部分参数名做了调整。

**&emsp;2、函数错误修复**：smc.Interpolate.NaturalNeighbor、rasp.Extraction.Reclassify、rasp.Surface.TRI、vesp.Basic.FixGeometry、vesp.Extraction.Split
